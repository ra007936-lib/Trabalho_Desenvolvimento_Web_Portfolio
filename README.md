## 📌 Sobre o projeto

Este projeto apresenta uma página simples e organizada, feita para treinar a base do desenvolvimento web.

Durante a criação, foram usados elementos de estrutura, estilo e interação com o usuário, deixando a página mais completa e visualmente melhor.

O objetivo principal é praticar como uma página web é montada, estilizada e como pode ter uma interação básica usando JavaScript.

---

## 🎨 Tecnologias utilizadas

| Tecnologia | Função no projeto |
|-----------|------------------|
| <strong>HTML</strong> | Criar a estrutura da página |
| <strong>CSS</strong> | Estilizar e organizar o visual |
| <strong>JavaScript</strong> | Adicionar interação na página |

---

## 🧠 O que foi praticado

| Conteúdo | Descrição |
|---------|-----------|
| <strong>Estrutura HTML</strong> | Organização dos textos, títulos e elementos da página |
| <strong>Estilização CSS</strong> | Uso de cores, espaçamentos, bordas e alinhamentos |
| <strong>Interação JS</strong> | Criação de uma ação simples com JavaScript |
| <strong>Organização</strong> | Projeto preparado para ser enviado no GitHub |

---

## 🚀 Objetivo da atividade

O objetivo da atividade é entender melhor como funciona a criação de uma página web, juntando a parte visual com a parte de interação.

Esse projeto ajuda a treinar conceitos importantes para quem está começando na área de desenvolvimento front-end.
